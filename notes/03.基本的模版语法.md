# 基本的模板语法

Vue.js 使用基于 HTML 的模板语法，能够声明式地将组件实例的数据绑定到渲染的 DOM 上。

## 文本插值

最基本的数据绑定形式，使用"Mustache"语法（双大括号）：

```html
<span>Message: {{ msg }}</span>
```

双大括号标签会被替换为相应组件实例中属性的值，且会随响应式数据的更改而同步更新。

## 原始 HTML

双大括号会将数据解释为纯文本，若想插入 HTML，需使用 `v-html` 指令：

```html
<p>使用文本插值: {{ rawHtml }}</p>
<p>使用 v-html 指令: <span v-html="rawHtml"></span></p>
```

**注意**：使用 `v-html` 时要确保内容安全，避免 XSS 漏洞。

## Attribute 绑定

在 HTML attribute 中无法使用双大括号，应使用 `v-bind` 指令：

```html
<div v-bind:id="dynamicId"></div>
```

### 简写语法

`v-bind` 的简写是 `:`：

```html
<div :id="dynamicId"></div>
```

## 使用 JavaScript 表达式

Vue 支持在数据绑定中使用 JavaScript 表达式：

```html
{{ number + 1 }}

{{ ok ? 'YES' : 'NO' }}

<div :id="`list-${id}`"></div>
```

注意：每个绑定仅支持单一表达式，不支持语句。

## 指令

指令是带有 `v-` 前缀的特殊 attribute，用于响应式地更新 DOM。例如 `v-bind` 用于绑定 attribute，`v-on` 用于监听事件。：

```html
<p v-bind:id="PID">Now you see me</p>
```

### 参数

某些指令需要参数，在指令后通过冒号隔开：

```html
<a v-bind:href="url"> ... </a>

<!-- 简写 -->
<a :href="url"> ... </a>

<a v-on:click="doSomething"> ... </a>

<!-- 简写 -->
<a @click="doSomething"> ... </a>
```
