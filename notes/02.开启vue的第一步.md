### 开启vue的第一步

首先们使用最朴素的方法来开始一个vue项目(有助于我们了解vue的运行机制)

##### vue的cdn地址
```
https://unpkg.com/vue@3/dist/vue.esm-browser.js
```

```html
   <script type="importmap">
  {
    "imports": {
      "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
    }
  }
</script>

<div id="app">
        <h1>{{message}}</h1>
    </div>
   
    <script type="module">
        import { createApp,ref } from 'vue'
        createApp({
            setup() {
                const message = ref("Hello world!")
                return {
                    message
                }
            }
        }).mount("#app")
    </script>

```


接下来可以尝试工程化来启动vue项目
```bash
pnpm create vue@latest
```
安装依赖包
```bash
pnpm i
```
创建项目成功后，进入项目目录，运行`pnpm dev`启动项目，
访问`http://localhost:5173/`即可看到项目启动成功。
