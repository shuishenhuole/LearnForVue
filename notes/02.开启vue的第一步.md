### 开启 vue 的第一步

首先们使用最朴素的方法来开始一个 vue 项目(有助于我们了解 vue 的运行机制)

##### vue 的 cdn 地址

```
https://unpkg.com/vue@3/dist/vue.esm-browser.js
```

```html
<script type="importmap">
  {
    "imports": {
      "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
    }
  }
</script>

<div id="app">
  <h1>{{message}}</h1>
</div>

<script type="module">
  import { createApp, ref } from "vue";
  createApp({
    setup() {
      const message = ref("Hello world!");
      return {
        message,
      };
    },
  }).mount("#app");
</script>
```

接下来可以尝试工程化来启动 vue 项目

```bash
pnpm create vue@latest
```

安装依赖包

```bash
pnpm i
```

依赖安装完成之后，运行`pnpm run dev`启动项目，
访问`http://localhost:5173/`即可看到项目启动成功。
