# 前置：环境搭建

在开始 Vue.js 学习之前，我们需要搭建好开发环境。本文档将详细介绍如何搭建 Vue.js 开发环境。

## 1. 安装 Node.js

Node.js 是运行 JavaScript 的环境，Vue.js 开发需要 Node.js 支持。

- [Node 官网地址](https://nodejs.org/zh-cn)
- 推荐安装 LTS（长期支持）版本
- 安装完成后，可以在终端中检查版本：

```bash
node --version
npm --version
```

## 2. 安装 pnpm 包管理器

pnpm 是一个快速、节省磁盘空间的包管理器，相比 npm 和 yarn 有更优的性能。

```bash
npm install -g pnpm@latest-10
```

安装完成后，检查版本：

```bash
pnpm --version
```

## 3. 安装代码编辑器

推荐使用 VS Code 作为代码编辑器，它有丰富的 Vue.js 插件支持。

- [VS Code 官网地址](https://code.visualstudio.com)
- 推荐使用VS Code，配合 Vue - Official 扩展 (之前是 Volar)。该插件提供了语法高亮、TypeScript 支持，以及模板内表达式与组件 props 的智能提示
